# Smriti Portfolio Starter

This is a simple static portfolio website you can open directly in a browser or run with a tiny local server.

## Step 1: Open the portfolio

You can double-click `index.html`, or run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Step 2: Update your personal content

Open `script.js` and edit the `portfolio` object at the top.

The most important fields are:

- `role`
- `summary`
- `about`
- `skills`
- `projects`
- `education`
- `achievements`
- `contact`

## Step 3: Replace placeholder project cards

Inside `script.js`, update the `projects` array with:

- project title
- what problem it solves
- what you built with
- what result or impact it had

## Step 4: Add your real education and achievements

Replace the starter text in:

- `education`
- `achievements`

Use details from your resume.

## Step 5: Update contact links

Replace these with your real links in `script.js`:

- email
- LinkedIn
- GitHub

## Step 6: Customize the design

Open `styles.css` if you want to change:

- colors
- spacing
- fonts
- card styles

The main color variables are at the top of the file under `:root`.

## Files

- `index.html`: page structure
- `styles.css`: design and layout
- `script.js`: all editable content
- `assets/Smriti-Pandey-Resume.pdf`: your resume download
